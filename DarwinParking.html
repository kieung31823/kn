Darwin Parking Smart App


Project Link

Description

This project aimed to develop Smart Parking App, an android-based mobile application that provides users with parking space information in Darwin city. The app should allow the users to select a parking zone and instantly see the total number of available parking spaces. The user should be able to choose a parking space and see if it is a car, disabled, or motorbike parking space and if it is free or paid. User should be able to select a parking space and see driving directions on the map. Dummy data was used instead of Darwin city councilâ€™s sensor data for parking space information like availability and fee.

Objectives

Enabling user to detect parking space availability in real-time and helping to optimize parking in cities.

- The accuracy and latency of a system are influenced by data from in-ground or overhead sensors helping detect vehicle presence and occupancy
